## Описание

Эта лабораторная работа содержит:
*   **Отчет** в файле `lab1.pdf` (исходники в файле `lab1.tex`).

*   **Метаморфное тестирование** в файле `meta.hs`.

## Запуск

### Фазз-тестирование

Для запуска фазз-тестирования:
```bash
ghc fuzzer.hs
./fuzzer
```

Для запуска метаморфного тестирования:
```bash
ghc meta.hs
./meta
```

При необходимости установки импортируемых модулей:
```bash
cabal update && cabal install random containers --lib
```
